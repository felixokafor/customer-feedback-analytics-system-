# customer-feedback-analytics-system-
Customer Feedback Analytics System (Google Forms + Sheets) ## Overview This project is a real-time data collection and analytics system built using Google Forms and Google Sheets. It captures customer feedback and transforms it into actionable business insights. --- ## Data Pipeline Google Forms → Google Sheets → Analytics Dashboard --- ## Features - Real-time data collection - Automated KPI calculations - Customer satisfaction tracking - Service performance analysis - Visual dashboard reporting --- ## Key Metrics - Average Satisfaction Score - Resolution Rate - Recommendation Rate - Service Efficiency Analysis --- ## Tools Used - Google Forms - Google Sheets - Excel-style formulas - Data visualization charts.
